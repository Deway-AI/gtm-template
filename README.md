Deway Web SDK Tag Template
This Google Tag Manager template allows you to easily integrate the Deway Web SDK into your website to track and identify users.

Features
Auto-Injection: Automatically loads the latest Deway SDK loader.

User Identification: Syncs User IDs directly from the GTM Data Layer.

Sandboxed Security: Built using GTM's secure Sandboxed JavaScript environment.

Setup Instructions
App Key: Obtain your App Key from the Deway Dashboard.

User ID Key: Provide the Data Layer variable name where your User ID is stored (e.g., user_id).

Trigger: Set the tag to fire on All Pages or Window Loaded.
